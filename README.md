# AI-Enhanced Intrusion Detection System (IDS)

Enterprise-grade, AI-enhanced IDS with real-time detection, classification, alerting, SOC dashboard, threat intel enrichment, and continuous learning.

This repository is being built in phases. See `TODO.md` for progress.

## Tech Stack
- Backend: Python, FastAPI, scikit-learn, TensorFlow/PyTorch, Kafka, Redis
- Data: PostgreSQL (relational), MongoDB (logs)
- Frontend: React, Tailwind CSS, Recharts
- Observability: Prometheus, Grafana
- Deployment: Docker, Kubernetes
